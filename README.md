# Redução dos Custos em Farmácias com AWS

Este desafio do Bootcamp Santander - DIO - 2025 tem como objetivo desenvolver um projeto para implementação de de ferramentas da nuvem AWS em uma empresa farmacêutica, onde o objetivo é reduzir os custos.
O relatório com as ferramentas e as suas funções e vantagens consta no arquivo .word nesse repositório. Também disponibilizei um arquivo com um exemplo da arquitetura projetada.
